# Rede Monitor

um sistema web de monitoramento de dispositivos de 
rede. Esse projeto é desafiador porque cada empresa 
registrada nele pode ter dezenas, centenas ou mais 
dispositivos (roteadores, servidores, etc) 
monitorados 24 horas por dia executando 
simultaneamente. O que demanda investimento 
em escalabilidade de microsserviços, principalmente 
o de monitoramento.
