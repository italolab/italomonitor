npm create vite@latest frontend-app --template react

npm install react-bootstrap bootstrap
npm install react-icons --save
npm install axios

npm install react-router-dom

npm install scss
npm install -D sass-embedded

npm install sockjs-client @stomp/stompjs
npm i --save-dev @types/sockjs-client

npm install date-fns --save

npm install recharts

Por causa do scss, é necessário adicionar isto ao defineConfig do arquivo vite.config.js:

css: {
    preprocessorOptions: {
      scss: {
        quietDeps: true
      }
    }
}

Por causa do sockjs-client, é necessário adicionar isto ao defineConfig do arquivo vite.config.js:

define: {
    global: 'window'
}
